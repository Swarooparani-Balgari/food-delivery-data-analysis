# Food Delivery Data Analysis

This project analyzes food delivery sales data using multiple tools.

Tools Used
- Excel
- SQL
- Python (Pandas)
- Power BI

Dataset Fields
- OrderID
- OrderDate
- Customer
- City
- Restaurant
- Category
- Quantity
- Price
- DeliveryFee

Analysis Performed
- Total Revenue
- Revenue by City
- Revenue by Category
- Top Customer
- Sales Trend


Key Insights
- Delhi generated the highest revenue
- Italian category generated the highest sales
- Rahul was the top customer
- Pizza Point had the highest revenue
- On the day 05 Jun, 2025 had the highest revenue
